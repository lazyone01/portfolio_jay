## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/lazyone01/portfolio_jay.git
```

Go to project folder:

```bash
cd portfolio_jay
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
